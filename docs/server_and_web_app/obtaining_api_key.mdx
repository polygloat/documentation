To obtain your API key, expand user menu and select API keys.

![img](../../static/img/docs/server_and_web_app/user_menu_api_key.png)

Then click ADD button. When dialog appears, select repository and check scopes you want to allow.
Then click GENERATE button.

![img](../../static/img/docs/server_and_web_app/api_key_generate_dialog.png)

Afterwards the newly created API key will appear in the list.
